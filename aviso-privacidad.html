PRIVACIDAD

Aviso de Privacidad

Explica qué información puede tratar Radar Laguna, para qué se utiliza y cómo funciona el consentimiento opcional de “Mi ubicación”.

Documento de trabajo para el lanzamiento público. Antes de publicar la aplicación, completa los datos definitivos del responsable del tratamiento y un medio de contacto para ejercer derechos de privacidad. Este texto no sustituye asesoría jurídica profesional.

1. Responsable y datos pendientes

Radar Laguna es un proyecto comunitario independiente. Antes del lanzamiento público deberá definirse y publicar de forma completa la identidad del responsable del tratamiento, así como el domicilio o medio de contacto aplicable y un canal para solicitudes relacionadas con datos personales.

2. Datos que puede tratar la aplicación

UbicaciónSolo cuando la persona pulsa “Mi ubicación”, lee la explicación y decide continuar. Se usa para centrar el mapa, detectar zona postal/colonia y facilitar un reporte.
ReportesEstado del agua, etiquetas seleccionadas, coordenada o punto del mapa, código postal, colonia/asentamiento cuando se confirma, fecha y hora.
Identificador técnico anónimoSupabase Auth puede crear una sesión anónima para aplicar seguridad, evitar votos duplicados y limitar abuso. No requiere nombre, correo ni contraseña.
Preferencias localesTema claro/oscuro, caché de reportes, consentimiento de ubicación y otros datos guardados en almacenamiento local del navegador.

3. Uso de “Mi ubicación”

Compartir la ubicación es completamente opcional. Radar Laguna muestra primero una explicación y permite elegir entre “No compartir” y “Continuar”. Solo después de continuar se solicita el permiso del navegador.

  • La ubicación se utiliza para una finalidad informativa y funcional dentro del radar.
  • Cuando se publica un reporte en Supabase, la coordenada se reduce a una precisión aproximada antes de almacenarse para disminuir la exposición de la ubicación exacta.
  • No se utilizará deliberadamente para publicidad, creación de perfiles, campañas políticas ni finalidades distintas a las informadas.
  • La persona puede rechazar el permiso y seguir usando el buscador o seleccionar manualmente un punto del mapa.
  • El permiso del navegador puede revocarse posteriormente desde la configuración del dispositivo o navegador.

4. Finalidades

  • Mostrar información comunitaria sobre disponibilidad y presión del agua.
  • Ubicar reportes en el mapa y asociarlos a una zona postal o colonia cuando sea posible.
  • Calcular señales comunitarias, confianza, tendencias y posibles cambios del servicio.
  • Mejorar la experiencia y funcionamiento de Radar Laguna.

5. Versión local actual y servicios técnicos de terceros

Cuando Supabase está habilitado, los reportes y votos se almacenan en una base de datos compartida para que la comunidad pueda ver actualizaciones en tiempo real. La app conserva además una caché local para mejorar la carga y funcionar de forma limitada durante fallas de red. Radar Laguna usa recursos técnicos externos como OpenStreetMap, Google Fonts y CDN de bibliotecas web; dichos servicios pueden recibir datos técnicos habituales de una solicitud web, como dirección IP, tipo de navegador o recursos solicitados.

6. Derechos y cambios futuros

Antes del lanzamiento público definitivo, este aviso deberá completarse para describir con precisión el responsable, plazos de conservación, transferencias, mecanismos de acceso, rectificación, cancelación y oposición, así como el procedimiento para revocar consentimiento y limitar el uso de datos.
